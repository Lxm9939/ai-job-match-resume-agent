建议截图尺寸：

- 截图前可使用 `examples/sample_resume.txt` 和 `examples/sample_jd.txt` 作为演示数据。
- Dashboard 截图建议使用 `examples/sample_jobs.csv` 或 V3 示例抓取结果，确保图表包含多个岗位。
