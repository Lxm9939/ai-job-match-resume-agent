# 截图说明

GitHub README 已预留截图位置。建议在本地启动 Streamlit 后，使用示例简历和示例 JD 完成一次分析，再手动截图并保存到本目录。

需要截图的页面：

- `home.png`：首页和输入区。
- `jd_parser.png`：JD 解析 tab。
- `evidence.png`：简历证据匹配 tab。
- `keywords.png`：关键词覆盖 tab。
- `score.png`：匹配评分 tab。
- `resume_optimization.png`：简历优化 tab。
- `outreach.png`：投递话术 tab。
- `final_report.png`：最终报告 tab。

建议截图尺寸：

- 桌面端：1440 x 900 或类似比例。
- 截图前可使用 `examples/sample_resume.txt` 和 `examples/sample_jd.txt` 作为演示数据。
