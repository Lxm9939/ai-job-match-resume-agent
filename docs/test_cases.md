## 回归检查清单

- `.env` 不应被提交。
- `.env.example` 不应包含真实 API Key。
- `.gitignore` 应排除 `__pycache__`、`.pytest_cache`、临时文件和本地导出报告。
- Streamlit 页面可启动。
- mock 模式下不依赖网络和 API Key。
